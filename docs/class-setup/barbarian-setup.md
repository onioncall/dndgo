# Barbarian Setup

### `path`

**Description:**
Path is the subclass for Barbarian. It can be any string and is not case sensitive

### `rage`

**Description:**
Barbarians have a tracked number of rages they can use before a rest.


**Fields:**
- `available`: int, current charges/tokens
- `maximum`: int, maximum charges/tokens
- `damage`: int, attack bonus to add to your melee weapons while raging

### `primal-knowledge`

**Description:**
When you reach 3rd level and again at 10th level, you gain proficiency in one skill of your choice from the list of skills available to barbarians at 1st level. Only use this if you are including *Tasha's Cauldron of Everything* as part of your campaign

**Allowed Values:** 
- "Animal Handling"
- "Athletics"
- "Intimidation"
- "Nature"
- "Perception"
- "Survival"
